## 📖 Описание

Этот мод воссоздаёт атмосферу **Level 0** из вселенной Backrooms — бесконечный лабиринт с жёлтыми стенами, влажным ковром на полу, гудящими люминесцентными лампами на потолке и жуткими существами, бродящими в коридорах.

### Что происходит в игре:

1. 🌍 Вы создаёте **обычный мир** и играете как обычно
2. ⏱️ Через **20 секунд** (настраивается) экран начинает **плавно темнеть**
3. 🌑 Экран полностью чернеет — вы **проваливаетесь**
4. 🏢 Вы **телепортируетесь** в отдельное измерение — **The Backrooms**
5. 🌕 Экран **плавно осветляется** — вы в лабиринте
6. 👾 В коридорах бродят **враждебные существа**

---

## 🗺️ Структура лабиринта

Лабиринт генерируется процедурно и **бесконечен** в любом направлении. Используется **4-уровневая система сеток**:

| Уровень | Размер сетки | Описание | Плотность |
|---------|-------------|----------|-----------|
| 🏛️ **Основные стены** | 32×32 блока | Огромные залы, основной каркас | 100% |
| 🚪 **Коридоры** | 16×16 блоков | Длинные коридоры, средние комнаты | 60% секторов |
| 🗄️ **Перегородки** | 8×8 блоков | Офисные кабинки в "плотных" зонах | 20% зон |
| 🏛️ **Колонны** | Каждые 10 блоков | Декоративные колонны в больших залах | 25% |

### Параметры мира:

- **Высота потолка:** 5 блоков (Y=1 — Y=5)
- **Пол (Y=0):** Неуязвимый (как бедрок, прочность −1)
- **Потолок (Y=6):** Неуязвимый + излучает свет (уровень 12)
- **Стены:** Прочные (hardness 50), но можно сломать инструментом

## 👾 Существа

В лабиринте спавнятся два типа враждебных существ:

### 🐺 Lurker (Скрытень)

| Параметр | Значение |
|----------|----------|
| HP | 20 ❤️ (10 сердец) |
| Урон | 6 ⚔️ (3 сердца) |
| Скорость | 0.38 (быстрее игрока!) |
| Дальность обнаружения | 48 блоков |
| Шанс спавна | 8% на чанк |

> **Тактика:** Lurker — быстрый охотник. Он бегает быстрее вас. Единственный шанс — спрятаться за стеной.

### 🦇 Howler (Воющий)

| Параметр | Значение |
|----------|----------|
| HP | 40 ❤️ (20 сердец) |
| Урон | 10 ⚔️ (5 сердец) |
| Скорость | 0.2 (медленный) |
| Броня | 4 🛡️ |
| Дальность обнаружения | 32 блока |
| Отбрасывание | 2.0 (сильное) |
| Шанс спавна | 4% на чанк |

> **Тактика:** Howler — медленный танк. Он не догонит вас на ровной поверхности, но один удар может быть смертельным. Избегайте тупиков!

## 🧱 Кастомные блоки

| Блок | ID | Прочность | Свет | Описание |
|------|----|-----------|------|----------|
| Стена Backrooms | `backrooms:backrooms_wall` | 50.0 | 0 | Жёлтая стена лабиринта |
| Пол Backrooms | `backrooms:backrooms_floor` | −1.0 ♾️ | 0 | Неразрушимый ковровый пол |
| Потолок Backrooms | `backrooms:backrooms_ceiling` | −1.0 ♾️ | 12 | Неразрушимый потолок с подсветкой |

> **Примечание:** Прочность −1.0 означает, что блок **невозможно сломать** никаким инструментом, как бедрок.

## 📦 Установка

### Требования

- **Minecraft** 1.20.1 (Java Edition)
- **Fabric Loader** 0.15.x или новее
- **Fabric API** 0.92.0+1.20.1
- **Java** 17 или новее

### Способ 1: Готовый мод (для игры)

1. Установите [Fabric Loader](https://fabricmc.net/use/installer/) для Minecraft 1.20.1
2. Скачайте [Fabric API](https://modrinth.com/mod/fabric-api) для 1.20.1
3. Скачайте `backrooms-1.0.0.jar` из [Releases](../../releases)
4. Скопируйте оба `.jar` файла в папку:
   ```
   %APPDATA%\.minecraft\mods\
   ```
5. Запустите Minecraft через лаунчер, выбрав профиль **fabric-loader-1.20.1**

### Способ 2: Из исходников (для разработки)

```bash
# Клонировать репозиторий
git clone https://github.com/binarnast80-cell/BackRooms_Mod_Minecraft.git
cd BackRooms_Mod_Minecraft

# Собрать мод
./gradlew build

# Запустить тестовый клиент Minecraft с модом
./gradlew runClient
```

Готовый `.jar` файл появится в `build/libs/backrooms-1.0.0.jar`

## 🎨 Кастомизация

### Замена текстур блоков

Замените PNG файлы (16×16 пикселей) в:
```
src/main/resources/assets/backrooms/textures/block/
├── backrooms_wall.png      ← Жёлтые обои
├── backrooms_floor.png     ← Ковровый пол
└── backrooms_ceiling.png   ← Потолочная плитка
```

### Замена текстур существ

Замените PNG файлы (64×64 пикселей, формат скина зомби) в:
```
src/main/resources/assets/backrooms/textures/entity/
├── lurker.png    ← Текстура Скрытня
└── howler.png    ← Текстура Воющего
```

### Добавление звуков

Добавьте `.ogg` файлы в:
```
src/main/resources/assets/backrooms/sounds/
├── lamp_hum.ogg          ← Гул ламп (loop)
├── ambient_backrooms.ogg ← Фоновый эмбиент (loop)
└── howler_cry.ogg        ← Крик Howler
```

> 💡 **Совет:** Бесплатные звуки можно скачать на [freesound.org](https://freesound.org)

## 🔧 Техническая информация

### Как работает телепортация

```
Наблюдение за игроком (ServerTickEvents)
        │
        ▼
  Отсчёт 400 тиков (20 сек)
        │
        ▼
  Состояние 0 → 1: Отправка пакета FADE_IN клиенту
        │
        ▼
  40 тиков (2 сек): Экран полностью чёрный
        │
        ▼
  Состояние 1 → 2: Телепортация в backrooms:the_backrooms
        │
        ▼
  20 тиков (1 сек): Пауза для загрузки чанков
        │
        ▼
  Состояние 2 → 3: Отправка пакета FADE_OUT клиенту
        │
        ▼
  40 тиков (2 сек): Экран осветляется
        │
        ▼
  Состояние 3 → 4: Завершено! Игрок в Backrooms
```

### Как работает генерация лабиринта

Генератор использует **детерминистическое хеширование** (не рандом), поэтому:
- ✅ Один и тот же chunk всегда генерируется одинаково
- ✅ Можно вернуться в ранее посещённое место
- ✅ Мультиплеер работает корректно

Алгоритм хеширования: `SplitMix64` — быстрый, качественный, без коллизий.
